This Page Gives a Short Description About The Web Application. Click the Earthquake Details Tab in the Navigation Bar above to Get the Data

  • This is a data driven web abblication which is developed using Python with Flask.Since the application is data driven application, we are displaying the data about EarthQuakes which are happened in the Month of February 2021. It shows the data like time when the Earthquake happened, longitute, latitute of the Earthquake and Magtype that calculates the magnitute of the event, locationsource the network that originally authored the reported location of this event etc.

  • We Created the application using two tables "mag_type" Table and "month_data" table which are linked on the basis of table field called 'MagType' field. When user clicks the Earthquake Detail tab on the navigation bar, it displays the page having dropdown list. In this page you can find all the relevant data about earthquakes on the basis of MagType. The dropdown loads all the MagTypes and when user selects the MagType and press the Submit Button , it displays the details on the basis of selected MagType.
  • Some MagTypes does not contain data because it calculates the earthquakes at higher range which luckily never happend like mww,mwc,ms,mfa, mi, mh. so in that case there prompts a message "No Data Available under This mgtype" You can get more than 5000 rows under 'ml' MagType and get morethan 2000 records under 'md' MagType. We also created the test cases in the test.py file to do the unit test of our application. We perform test cases on MgType field with the help of test funtions and then we check input for getMgDetails() Function.
